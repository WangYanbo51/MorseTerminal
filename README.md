# MorseTerminal

一个基于 **Pygame** 与 **ModernGL (OpenGL 3.3)** 构建的复古风摩尔斯电码终端模拟器。本项目通过精心编写的 GLSL 着色器，不仅能实现文本与电码的实时动态编解码、方波绘制，还完美模拟了早期 CRT 监视器的硬件物理特性，如荧光屏磷光体残影余晖（Phosphor Persistence）、球面畸变、扫描线、色差和屏幕色散。

---

## ✨ 核心特性

*   **智能 WPM 动态学习**：系统能根据你敲击空格键的节奏，动态计算并学习你的平均输入速度（WPM），自动调整断字（Character Timeout）与断词（Word Timeout）的判定阈值。
*   **硬件级 CRT 模拟 (GLSL Shader)**：
    *   **球面畸变与暗角**：模拟老旧微显像管的物理弯曲边缘与四周光衰减。
    *   **动态荧光残影**：采用 GPU 帧缓冲双缓冲技术，根据帧间隔（$\Delta t$）指数级平滑衰减，还原高逼真的绿光电子束余晖。
    *   **色差色散 (Chromatic Aberration)**：模拟电子枪对焦不准导致的红绿蓝（RGB）边缘溢出。
    *   **滚动的扫描条纹与闪烁**：为画面注入复古的“微弱不稳定感”。
*   **高帧率自适应位移**：底部的方波信号采用独立于帧率（Delta-time base）的亚像素位移逻辑，无论显示器是 60Hz 还是 144Hz+，波形滚动的视觉速度始终保持一致，不会发生高刷屏下波形暴走的情况。
*   **系统级鲁棒性优化**：
    *   在 Windows 平台下原生适配高 DPI 缩放，避免全屏切换导致的鼠标热区错位。
    *   运行期间自动 associate 并挂起 IME 输入法，防止中文输入法弹窗中断电码录入。

## 🛠️ 环境依赖与安装

项目依赖 C++ 编译器底层支持的 `ModernGL` 图形库和 `Pygame` 音频/事件库。

### 1. 确保安装 Python
推荐使用 Python 3.8 或更高版本。

### 2. 安装依赖库
打开终端执行以下命令安装所需模块：
```bash
pip install pygame moderngl
